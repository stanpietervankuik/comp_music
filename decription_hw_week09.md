# Clocks (Coldplay) - Chroma vs Timbre (Week 09)

Clocks repeats the same piano pattern and chord loop for most of the song. 
You can see that in the chromagrams: the same few pitch classes keep showing up again and again across the track. 
The chroma self-similarity matrix also shows lots of similarity over time, which fits with the harmony staying mostly the same. 
The biggest exception is around **2:32**, where there is a clear break and the piano plays different notes/chords.

The timbre plots show section changes more clearly. The MFCC cepstrogram stays fairly consistent overall, 
but it shifts when the arrangement changes, like when **drums first enter (0:21)**, **vocals enter (0:35)**, **heavier drums come in (1:20)**, 
and during the **break at ~2:32**. The timbre self-similarity matrix also has clearer “blocks”,
meaning the overall sound/texture changes between these sections even though the chord loop often continues underneath.
